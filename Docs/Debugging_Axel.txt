Skriver upp vad registret har för värde när man går ut ur sektionen
--JOY_<DIR>
R0 - DIRECTION
R4 - DELTA Y
R5 - DELTA X
RF - 1 - very important to have this!

--MOVE
R1 - X_POS
R2 - Y_POS
R6 - DRILL_LEVEL
R7 - NEW_X_POS
R8 - NEW_Y_POS
RE - NEW CONSTANT

--NEXT_SCREEN
RE - NEW CONSTANT

--X/Y_WRAP_<DIR>
RC - CONST
RB - TILE_ADRESS FOR TILE IN NEW ROOM
RA - TILE FROM NEW ROOM
RE - HIGHEST BREAKABLE TILE
RD - BREAKABLE BLOCKS
R5 - NEW_WRAP_X
R6 - NEW_WRAP_Y
R9 - START ADRESS FOR NEW ROOM TILES IN DMEM

--CHANGE SCREEN (Score calculation shenanigans only)
RE - CONST
RA - SCORE_CALC
RD - SCORE

--CONT_CHANGE_SCREEN
RE - CONST
R2 - ADRESS FOR PLAYER IN VMEM
RD - GROUND_TILE ADRESS
R3 - CONST
R4 - START_ADRESS FOR CUR ROOM IN DMEM
RA - LOOP X
RE - LOOP Y
RD - CONST 1
RC - CONST 8

--CHANGE_SCREEN_LOOP
R0 - 20
R1 - Y * 20
R1 - X + 20*Y
R3 - ADRESS FOR STORING CURRENT TILE (X + Y*20 + DMEM_START_CUR_ROOM)
R2 - Y + 1 (STATUS BAR OFFSET)
R2 - LSL 8 (FOR ADRESSING VMEM)
R2 - R2 + X FOR TILE IN VMEM
R5 - CURRENT TILE
R0 - 19
